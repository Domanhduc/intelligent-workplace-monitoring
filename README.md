# Intelligent Workplace Monitoring System

## Overview
This project presents an AI-based intelligent workplace monitoring system using ESP32-CAM and YOLOv8 for real-time object detection and environment supervision.

The system integrates embedded vision, edge computing, and network communication to provide an efficient smart monitoring solution.

## Key Features
- Real-time image acquisition using ESP32-CAM  
- YOLOv8 object detection  
- MQTT-based communication  
- IoU-based performance evaluation  
- Embedded and edge AI integration  

## System Architecture
ESP32-CAM → MQTT → Server → YOLOv8 → Monitoring Interface

## Technologies
- Python  
- ESP32-CAM  
- YOLOv8  
- MQTT  
- Computer Vision  

## Results
The system successfully detects and monitors target objects in real time with reliable performance in indoor environments.
